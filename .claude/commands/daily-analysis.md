# Инстркуции
Проанализировать портфолио, вывести сыьлные ислады места, да рекомендации. Напиши репорт очень кратко.


## Задачи
- прочитай новостную сводку за послении 24 часа: $ARGUMENTS
- Используй reports/Sample.md n reports/Template.md как образец репорта
- Запустить "bash uv run python -m src.cli summary"
- Запустить "bash uv run python -m src.cli cash"
- Запустить "bash uv run python -m src.cli allocation"
- Проанализировать и написать репорт
- Созарнить репорт в папку "reports/"
- Формат репорта .md, паттерн названия: {yyyy-MM-dd}-{Остроумное_название_на_русском}.md
- Язык: Русский
