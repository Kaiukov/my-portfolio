## Noise Pattern Recognition Framework

### Red Flags for Noise Headlines
- ✗ Single entity action (whale, CEO, influencer) without corroboration
- ✗ Event attendance without announcements
- ✗ Political opinions or predictions
- ✗ Regulatory ambiguity (disputes without resolution)
- ✗ Price movements at conferences (revert quickly)
- ✗ VC funding without demonstrated user adoption
- ✗ On-chain activity without attribution to known institutions
- ✗ Unverified on-chain tracking claims

### Red Flags for POTENTIAL Signals
- ✓ Multiple independent sources confirming same information
- ✓ Official announcements from verified institutions
- ✓ Regulatory clarity (court ruling, legislation passed)
- ✓ Demonstrated adoption metrics (users, TVL, transactions)
- ✓ Institutional investor official commitments
- ✓ Macro structural changes (interest rates, policy, technology)
- ✓ Competitive advantage shifts (market share changes)
- ✓ Consensus expert opinion (multiple analysts, multiple publications)
